 Analizador de Sentimientos - Proyecto #4

Este proyecto realiza análisis de sentimientos utilizando VADER, un modelo preentrenado incluido en la librería NLTK.

Forma parte de mi desafío de desarrollar 100 proyectos relacionados con Inteligencia Artificial.

---

 Tecnologías utilizadas

- Python
- NLTK
- VADER Sentiment Analyzer
- Procesamiento de texto

---

 ¿Cómo funciona?

El programa:

- Recibe una frase ingresada por el usuario
- Limpia la puntuación
- Utiliza el modelo VADER para calcular puntajes de sentimiento
- Clasifica el resultado como positivo, negativo o neutro

 Nota: El modelo funciona mejor con frases en inglés.
